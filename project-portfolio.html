个人作品集网站

2026-04-18 UI设计 阅读次数: 5,432

项目简介

使用 HTML5 + CSS3 构建的现代化个人作品集网站，展示个人项目、文章和技术博客。

功能模块

  • 首页：瀑布流文章卡片展示、分类筛选、英雄区域
  • 项目展示：项目案例详情、技术栈展示、GitHub链接
  • 关于我：个人介绍、技术栈、技能展示
  • 文章详情：Markdown渲染、代码高亮、阅读统计
  • 联系页面：联系表单、社交媒体链接

技术栈

HTML5
CSS3
JavaScript
Font Awesome
Google Fonts

设计特点

个人作品集网站界面预览

项目亮点

  1. 响应式设计，完美适配桌面端、平板、手机
  2. 毛玻璃效果（backdrop-filter）增强视觉层次
  3. 纯 CSS 动画效果（过渡、变换、关键帧动画）
  4. 语义化 HTML5 标签，提升 SEO 和可访问性
  5. Flex 布局实现圣杯布局，侧边栏固定+主内容滚动
