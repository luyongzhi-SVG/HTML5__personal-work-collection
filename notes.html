一、HTML5 新特性

1.1 语义化标签

HTML5引入了一系列语义化标签，使页面结构更加清晰，提升SEO和可访问性。

标签 用途
<header> 页面头部，包含标题、导航等
<nav> 导航区域，网站导航链接
<section> 内容区块，语义化的内容分组
<article> 独立文章，可独立发布的内容
<aside> 侧边栏，辅助信息区域
<footer> 页脚，版权信息等

1.2 多媒体标签

HTML5原生支持音视频播放，无需第三方插件。

<!-- 音频 -->
<audio src="music.mp3" controls>您的浏览器不支持音频</audio>

<!-- 视频 -->
<video src="video.mp4" controls width="640" height="360">
    您的浏览器不支持视频
</video>

新增多种输入类型和属性，提升用户体验和表单验证。

二、CSS3 新特性

2.1 选择器

CSS3扩展了选择器功能，提供更强大的样式控制能力。

/* 属性选择器 */
input[type="text"] { ... }

/* 伪类选择器 */
a:hover { ... }
li:nth-child(2n) { ... }

/* 伪元素选择器 */
p::first-line { ... }
p::before { content: ""; }

2.2 盒模型与边框

.box {
    box-sizing: border-box;  /* 宽高包含内边距和边框 */
    border-radius: 8px;      /* 圆角 */
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);  /* 阴影 */
}

2.3 背景与渐变

/* 线性渐变 */
.gradient {
    background: linear-gradient(45deg, #007bff, #0056b3);
}

/* 径向渐变 */
.radial {
    background: radial-gradient(circle, #ff0000, #0000ff);
}

/* 多重背景 */
.bg {
    background: url(img1.jpg) no-repeat left top,
                url(img2.jpg) no-repeat right bottom;
}

2.4 文本效果

.text {
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);  /* 文字阴影 */
    word-break: break-all;  /* 换行处理 */
    text-overflow: ellipsis;  /* 省略号 */
}

三、CSS3 动画与过渡

3.1 过渡（Transition）

实现元素状态变化时的平滑过渡效果。

.box {
    transition: all 0.3s ease;  /* 属性 时长 缓动函数 */
}

.box:hover {
    transform: scale(1.1);
}

3.2 动画（Animation）

使用@keyframes定义动画关键帧。

@keyframes spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

.loading {
    animation: spin 1s ease-in-out infinite;
}

3.3 变换（Transform）

.transform {
    transform: translate(10px, 20px);  /* 位移 */
    transform: rotate(45deg);           /* 旋转 */
    transform: scale(1.5);              /* 缩放 */
    transform: skew(10deg, 5deg);       /* 倾斜 */
    transform-origin: center center;    /* 变换原点 */
}

四、响应式设计

4.1 媒体查询（Media Queries）

根据设备特性应用不同样式。

/* 基础样式 */
.container { width: 1200px; }

/* 平板设备 */
@media (max-width: 1024px) {
    .container { width: 90%; }
}

/* 手机设备 */
@media (max-width: 768px) {
    .container { width: 100%; }
}

4.2 视口设置

<meta name="viewport" content="width=device-width, initial-scale=1.0">

4.3 弹性布局（Flexbox）

.flex-container {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
    align-items: center;
}
提示：采用移动优先原则，先设计移动端布局，再逐步扩展到更大屏幕。

五、高级 CSS 效果

5.1 毛玻璃效果

.glass {
    background: rgba(255, 255, 255, 0.1);
    backdrop-filter: blur(10px);
    -webkit-backdrop-filter: blur(10px);
}

5.2 滤镜（Filter）

.img-bright { filter: brightness(1.2); }    /* 亮度 */
.img-blur { filter: blur(5px); }            /* 模糊 */
.img-grayscale { filter: grayscale(50%); }  /* 灰度 */

5.3 文字渐变

.gradient-text {
    background: linear-gradient(45deg, #007bff, #6610f2);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
}

六、学习路径与最佳实践

6.1 学习路径

  1. 基础阶段：HTML5语义化标签、CSS3基础选择器和属性
  2. 进阶阶段：CSS3动画、过渡、变换
  3. 实战阶段：响应式设计、综合项目开发
  4. 高级阶段：性能优化、浏览器兼容性处理

6.2 最佳实践

  • 语义化优先：使用正确的HTML5标签
  • 样式分离：CSS写在单独文件中
  • 响应式设计：移动优先原则
  • 代码规范：统一缩进、注释清晰
  • 性能优化：图片压缩、CSS精简
