技术博客平台

2026-04-16 前端项目 阅读次数: 3,789

项目简介

基于 Next.js 开发的现代化博客平台，支持 Markdown 编辑、代码高亮、评论系统等功能。

功能特性

  • 文章管理：Markdown 编辑器、实时预览、分类标签
  • 评论系统：支持回复、点赞、嵌套评论
  • 用户系统：注册登录、用户中心、权限管理
  • 搜索功能：全文搜索、标签筛选、热门文章
  • 主题切换：支持亮色/暗色主题切换

技术栈

Next.js
React
TypeScript
Tailwind CSS
Prisma
PostgreSQL

项目亮点

  1. SSR/SSG 混合渲染，SEO 友好
